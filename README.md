# Test 03MSEC - Sociedad, Familia y Educación

Repositorio con un test interactivo de repaso para la asignatura **Sociedad, Familia y Educación** del MAES.

El objetivo es facilitar el estudio mediante preguntas tipo test basadas en bancos de años anteriores y organizadas en formato interactivo para practicar de forma autónoma.

## Contenido

- Test interactivo en HTML.
- Preguntas de años anteriores depuradas.
- Banco de preguntas limpio y renumerado.
- Eliminación de duplicados evidentes.
- Respuestas correctas unificadas.
- Modo de práctica para repasar errores.

## Archivos principales

- `index.html`: test interactivo principal.
- `banco_limpio_03MSEC.md`: banco de preguntas limpio en formato Markdown.
- `banco_limpio_03MSEC.json`: banco de preguntas estructurado.
- `informe_limpieza_03MSEC.md`: resumen del proceso de limpieza del banco.

## Cómo usarlo

Abrir el archivo `index.html` directamente en el navegador o acceder mediante GitHub Pages.
